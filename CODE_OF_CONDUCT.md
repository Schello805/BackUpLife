# Code of Conduct

Dieses Projekt soll sachlich, freundlich und respektvoll gepflegt werden.

## Erwartetes Verhalten

- respektvolle Kommunikation
- konstruktives Feedback
- Fokus auf Sicherheit, Klarheit und Nutzbarkeit

## Nicht akzeptabel

- beleidigendes oder herabwürdigendes Verhalten
- Veröffentlichung sensibler Zugangsdaten
- absichtliches Einbringen unsicherer oder irreführender Funktionen
